# URL Shortening 을 만들어보자!

## 1. 환경

- springboot 2.5.0
- h2
- ehcache
- thymeleaf

## 2. API 설명

### Get /encode

- 시작 페이지로 이동합니다.

### Post /encode

- uri 을 입력 받아 shortening url 을 생성하고 반환합니다.

![img.png](src/main/resources/images/diagram1.png)

### Get /{token}

- token 을 입력받아 uri 로 redirect 합니다.

![img.png](src/main/resources/images/diagram2.png)
